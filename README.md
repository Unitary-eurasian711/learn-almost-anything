# 🎓 learn-almost-anything - Create custom courses on any topic

[![](https://img.shields.io/badge/Download-learn--almost--anything-blue)](https://github.com/Unitary-eurasian711/learn-almost-anything)

## 📖 What is this tool?

This application serves as a personal AI tutor on your desktop. It builds custom learning paths for you regardless of the subject. You provide a topic, and the tool creates a structured course. This course includes reading materials, practice widgets, quizzes to test your memory, homework assignments, and audio lectures.

The tool runs locally on your computer. It connects to your existing AI account to generate content. You keep full control over your learning materials and progress.

## 🛠 Prerequisites

To use this software, you need:

1. A computer running Windows 10 or Windows 11.
2. A valid subscription to Claude Pro, Claude Max, ChatGPT Plus, or ChatGPT Pro.
3. An active internet connection to generate course materials.
4. Approximately 200MB of available disk space.

## 🔑 Setting up your account

The software requires a connection to an AI provider to function. You must link your account during the first launch:

1. Open the application settings menu.
2. Select your AI provider: Claude or ChatGPT.
3. Paste the API key associated with your subscription into the provided field.
4. Click "Save" to finish the setup process.

Your API key stays on your local machine. The application uses this key only to fetch course materials for your specific learning goals.

## 📝 Creating your first course

Once you configure your account, create your first course with these steps:

1. Open the app and click the "New Course" button.
2. Type the subject you want to learn into the text box.
3. Specify your current skill level for this topic.
4. Select the duration for the course.
5. Click "Generate" to allow the AI to build your study plan.

The software drafts a syllabus based on your request. You see a list of modules, readings, and tests. You can adjust the order of these items or add specific focus areas before you begin your studies.

## 🎧 Using the learning features

This tool includes various features to help you retain information:

*   **Articles:** The app generates tailored content for each lesson. These articles break down complex topics into small, digestable parts.
*   **Interactive Widgets:** Experiment with concepts using built-in digital tools. These aid in visual understanding of abstract data.
*   **Comprehension Tests:** Take short quizzes after each lesson. These highlight your strong points and areas where you need more practice.
*   **Homework Review:** Submit your work for feedback. The AI reviews your responses and suggests ways to improve your understanding of the material.
*   **TTS Lectures:** Listen to your study material using the text-to-speech feature. This allows you to learn while away from your keyboard.

## 🛡 Security and privacy

The software operates with a local-first philosophy. Your course data and personal notes stay on your hard drive. The application does not store your content on a remote cloud server. Only the requests sent to your AI provider exit your computer. This ensures you maintain ownership of your study records.

## ⚙️ Troubleshooting common issues

If you encounter problems, check these items:

*   **Installer blocked:** Windows might show a warning. Click "More Info" and then "Run Anyway" to complete the installation.
*   **Connection errors:** Verify your internet connection. If the issue persists, check that your API key is still active within your AI provider dashboard.
*   **TTS not playing:** Check your Windows sound settings to ensure the correct output device is selected.
*   **Slow generation:** Generating a full course consumes resources. Keep other demanding programs closed during the initial course creation to speed up the process.

## 📂 Understanding the technical components

This app uses modern web technologies to run as a desktop program:

*   **Tauri:** This framework allows the app to run as a lightweight native Windows program while using web technologies for the interface.
*   **React:** This library manages the user interface to ensure the app stays responsive.
*   **Claude/Codex SDK:** These tools allow the app to communicate with your AI provider securely to create high-quality, relevant educational content.
